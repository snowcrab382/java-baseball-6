숫자야구 기능명세서 V2

- [v]게임을 시작한다
- [v]컴퓨터의 숫자를 생성한다
    - 숫자로 이루어져야 한다
    - 3자리 숫자여야 한다
    - 각각의 자릿수가 모두 달라야한다
    - 0이 있으면 안된다
- [v]유저의 숫자를 입력받는다
    - 숫자로 이루어져야 한다
    - 3자리 숫자여야 한다
    - 각각의 자릿수가 모두 달라야한다
    - 0이 있으면 안된다
- [v]컴퓨터와 유저의 숫자를 비교한다
- [v]비교 결과를 반환한다
- [v]비교 결과를 합산해 출력한다
- [v]최종 결과를 확인한다
- [v]게임 재시작 또는 게임 종료를 결정한다
